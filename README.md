# Portfolio Website

A modern, responsive portfolio website built with React, Vite, and Tailwind CSS. This portfolio showcases your work, skills, and provides a way for potential clients or employers to contact you.

## Features

- Responsive design that works on all devices
- Smooth animations using Framer Motion
- Modern UI with Tailwind CSS
- Contact form
- Project showcase
- Skills section
- Social media integration

## Getting Started

1. Clone the repository:
```bash
git clone <your-repo-url>
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Customization

1. Update your personal information in the respective components:
   - `src/pages/Home.jsx` - Update your name and introduction
   - `src/pages/About.jsx` - Update your background and skills
   - `src/pages/Projects.jsx` - Add your projects
   - `src/pages/Contact.jsx` - Update social media links

2. Replace placeholder images with your own project screenshots

3. Customize the color scheme in `tailwind.config.js`
